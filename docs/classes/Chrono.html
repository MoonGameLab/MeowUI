chrono\create (owner, duration, repeated, alwaysUpdate, onDone)
creates a timer. This timer will be removed automatically from owner if "chrono" is the key for the timer instance. You can set it to be repeated with the same duration and onDone callback.

Parameters:

  • owner Control
  • duration number
  • repeated boolean
  • alwaysUpdate bool
  • onDone function
